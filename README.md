# TicTacToeOnline
This is a project to bring the hit childerens' game, tic tac toe, to the modern age!


# Features
- Play from any device
- Clean UI
- No ads
- Light weight
- Self hosted
- Totally anonymous

# How to run
Install java version 17 or greater

Run the following command:

```
java -jar <filename>.jar
```
